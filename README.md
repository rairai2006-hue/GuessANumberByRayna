# GuessANumberByRayna
This is a simple console game "Guess a number"
## How to play?
1. The computer generates a random number between one and a hundred.
2. Then the player should type their guesses. After each guess he will be shown whether his guess is lower tor higher than the computer's chosen number
3. The player has only ten tries.

### Technologies
For the making of this simple console game is used the programming language Python.
